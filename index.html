Projeto pessoal · Finalizado

Finanças Pessoais

Sistema web desenvolvido para controle financeiro pessoal, integrado ao Firebase, com login e cadastro por e-mail e senha, salvamento online de dados, dashboard mensal, páginas separadas por área, cadastro de transações, contas a pagar, total em aberto, alertas automáticos de vencimento e alarme visual/sonoro dentro do próprio site.

HTML5 CSS3 JavaScript Firebase Auth Cloud Firestore Responsividade

Dados salvos online para acessar pelo celular, PC ou notebook
Dashboard mensal com entradas, saídas, saldo e total em aberto
